# Участие в разработке

Спасибо за интерес к проекту Lango! Это учебный проект, но pull request-ы с исправлениями и улучшениями приветствуются.

## Как внести вклад

1. Сделайте форк репозитория
2. Создайте ветку для своей доработки: `git checkout -b feature/моя-фича`
3. Внесите изменения и зафиксируйте их с понятным сообщением коммита
4. Отправьте ветку в свой форк: `git push origin feature/моя-фича`
5. Откройте Pull Request в ветку `main`

## Стиль кода

- Следуйте [соглашениям Kotlin](https://kotlinlang.org/docs/coding-conventions.html)
- Придерживайтесь лучших практик Jetpack Compose (stateless-компоненты там, где это возможно)
- ViewModels не должны зависеть от Android-фреймворка там, где можно обойтись без этого
- Перед коммитом отформатируйте код: **Android Studio → Code → Reformat Code**

## Сообщить об ошибке

Откройте GitHub Issue и укажите:
- Версию Android и модель устройства / эмулятора
- Шаги для воспроизведения
- Ожидаемое и фактическое поведение
- Вывод logcat (если есть)
